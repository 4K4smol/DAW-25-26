UD3E2 – Depurador visual (DOM)

Ejemplo de contenido

Esta página sirve como escenario de pruebas para el depurador visual del DOM. Podrás ir navegando entre nodos padre, hijos y hermanos.

Selecciona distintos elementos (por ejemplo desde el código) y comprueba cómo el depurador resalta el nodo actual y muestra su información.

Lista de tareas

  • Implementar la clase NodoInspector.
  • Crear el panel flotante de depuración dinámicamente.
  • Aplicar el estilo de resaltado al nodo seleccionado.
